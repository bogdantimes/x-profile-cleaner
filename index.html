FAQ

Is this safe to use?

Yes. The script runs entirely in your browser and doesn't send any data to external servers. It's open-source so you can review the code yourself.

Can I recover deleted tweets?

No. Once deleted, tweets cannot be recovered. Download your Twitter archive first if you want a backup.

Will this get me banned?

The script uses normal browser actions (clicking buttons) and includes delays to respect rate limits. However, use responsibly.

Why do I need to enter my username?

The script needs to identify your tweets vs. tweets from others that appear on your profile (replies, mentions, etc.).

Can I delete tweets from a specific time period?

Yes! Use X's advanced search to filter by date. Search for from:yourusername since:2023-01-01 until:2024-01-01 and run the script on those search results. The script will only delete tweets visible in that filtered view. You can also use other search operators like filter:replies, min_faves:10, or keyword searches to target specific tweets.

Can I delete only replies or only tweets with certain keywords?

Yes! Use X's search operators: from:yourusername filter:replies for only replies, or from:yourusername "keyword" for tweets containing specific words. Run the script on the search results page.

It's stuck, what do I do?

Click "Stop", refresh the page, and try again. Some tweets (like pinned tweets) may need manual deletion.
